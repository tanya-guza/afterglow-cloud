AfterGlow Cloud is a security visualization tool which lets you upload data and visualize the data as graphs on-the-fly. You get to choose over the specifics, settings of the graph and generate them real time.

Render a graph

Open-source

AfterGlow Cloud is an open-source project and is based on its command line predecessor AfterGlow

Different input formats

AfterGlow Cloud allows visualizing and analyzing data from different sources - CSV, raw log files and Loggly logging service.

Interactive

You can analyze and manipulate the data in the real-time by using simple but yet powerful interactive tools
